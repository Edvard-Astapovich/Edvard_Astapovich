Блок навигации

  • Ширина 15em
  • Должен присутствовать цвет фона
  • Располагается слева
  • Блок навигации наполните контентом, достаточным, что бы по высоте он начал захватывать вторую статью, как показано в данном примере

Статья 1

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех статьях сделайте отступ снизу равным .2em, при помощи свойтва margin-bottom

Статья 2

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех статьях сделайте отступ снизу равным .2em, при помощи свойтва margin-bottom. Картинка втавлена при помощи тэга, плавание организовано по правому краю внутри дaнной статьи для этой картинки.

Нижний блок. Подвал. Занимает всю доступную ширину
